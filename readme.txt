Integrates Nextgen managed images, galleries and albums with your Wiziapp powered iphone app.

== Description ==

This plugin integrates with your Wiziapp powered iPhone App. 

WiziApp is a plugin that turns your WordPress powered web site into a native iPhone App. [Click here](http://wordpress.org/extend/plugins/wiziapp-create-your-own-native-iphone-app/) for more information. 

This plugin collects all your Nextgen images, Galleries and albums and add it to your Wiziapp powered iPhone App Albums. 

To learn more about Wiziapp albums feature, visit the [Wiziapp plugin tour](http://www.wiziapp.com/tour) .

The Wiziapp to Nextgen plugin works in conjunction with the Wiziapp [install now](http://wordpress.org/extend/plugins/wiziapp-create-your-own-native-iphone-app/) & the Nextgen [install now](http://wordpress.org/extend/plugins/nextgen-gallery/) plugins.
